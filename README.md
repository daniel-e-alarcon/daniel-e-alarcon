## Featured Projects
### RAG Explorer (Streamlit)

A local-first Retrieval-Augmented Generation (RAG) application for document Q&A with cited sources and reproducible exports. Includes FAISS index management, provider toggle (Ollama default, optional OpenAI), hybrid retrieval (BM25/Dense), and retrieval evaluation (hit@k, MRR) across QA sets.

Repo: https://github.com/daniel-e-alarcon/rag-explorer

### AI Data Insight Dashboard (Streamlit)

Interactive dashboard for CSV-based EDA and baseline ML modeling, including cross-validation, confusion matrix/ROC (classification), residual diagnostics (regression), feature importances, and structured LLM reporting (local-first with optional OpenAI fallback).

Repo: https://github.com/daniel-e-alarcon/ai-data-dashboard
